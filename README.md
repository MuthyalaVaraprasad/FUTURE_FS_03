# 🏋️ Fitness Club

A modern and responsive **Fitness Club Web Application** built with **React.js**, **Material UI**, and the **ExerciseDB API**. This application helps users discover exercises, search by muscle group or exercise name, and view detailed workout information with animated GIFs.

## 🌐 Live Demo

🔗 https://fitnessclub-six.vercel.app/

---

## 📖 Overview

Fitness Club is designed to make fitness learning simple and interactive. Users can search thousands of exercises, explore workouts by body part, and access detailed information including target muscles, equipment, and exercise demonstrations.

---

## ✨ Features

- 🔍 Search exercises by name
- 💪 Browse exercises by body part
- 🏋️ View exercise details
- 🎯 Target muscle information
- 🛠 Equipment information
- 🎥 Animated exercise demonstrations (GIFs)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/MuthyalaVaraprasad/PRODIGY_WD_02.git
```

Move into the project folder

```bash
cd PRODIGY_WD_02
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```

Open your browser

```
http://localhost:3000
```
